PropertyPulse is a real-estate web app where users can come and browse for properties, users can search, message, bookmark and share the property they like. This project is built with performance, scalability and clean UI. 

# Live Demo 
Link: https://property-pulse-delta.vercel.app/

## Features: 
- Browse property listings
- Search properties
- Add filters during searching
- Sign in using google account
- Add a property
- Bookmark a property
- Share a property
- Contact or Message the property owner
- Check out your profile
- Remove a message
- Mark a message as read/unread
- Featured properties
- Notifications
- Edit a property ( for owners )
- Remove a property ( for owners )
- Pagination


## Tech Stack
- Framework: Next.js (App Router)
- Language: Typescript
- Styling: TailwindCSS
- Database: Mongodb
- Database Connection: mongoose
- Map: MapLibre / React Map GL ( library )
- Images: Used cloudinary for converting images into urls
- Deployment: Vercel

## Getting Started

1. Clone the repository
   - git clone https://github.com/M-Jawed/PropertyPulse
   - cd property-pulse
2. Install dependencies
   - npm install
3. Run the development server
   - npm run dev

## Env variables
- MONGODB_URI=Mongodb URI
- GOOGLE_CLIENT_ID=Google client id (oAuth)
- GOOGLE_CLIENT_SECRET= Google client secret (oAuth)
- NEXTAUTH_SECRET=Nextauth secret (random long words)
- CLOUDINARY_CLOUD_NAME=Cloudinary cloud name
- CLOUDINARY_CLOUD_API=Cloudnary api
- CLOUDINARY_CLOUD_SECRET=Cloudinay secret
- NEXT_PUBLIC_GEOAPIFY_API=Geoapify API
